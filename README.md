# Salt Flat Coffee

Marketing site for Salt Flat Coffee Company, a small-batch artisan coffee roaster based in Tooele, Utah. The site serves as both a brand presence and product showcase, built with performance and visual polish in mind. Zero component libraries, all custom CSS.

Live at [saltflat.andenwick.dev](https://saltflat.andenwick.dev)

## Features

- **Scroll-driven animations** - IntersectionObserver-based reveal system (`ScrollReveal`) with configurable delays, threshold tuning, and a 2-second fallback timer to guarantee content visibility
- **Procedural SVG wave dividers** - Section transitions generated by `generate-waves.js`, producing unique organic curves between dark/grey/light sections without static assets
- **Scroll progress line** - Procedural SVG path (`ScrollProgress`) that draws along the page as the user scrolls, with rAF-throttled updates and a checkpoint indicator at the end
- **Hover-to-reveal product labels** - Product cards crossfade from the product bag image to the label artwork on hover using CSS opacity transitions
- **rAF-throttled scroll handlers** - Header and scroll progress both gate scroll listeners behind `requestAnimationFrame` to avoid layout thrashing
- **Fullscreen mobile navigation** - Animated burger menu with staggered link transitions, body scroll lock, and overlay fade
- **Newsletter signup form** - Email capture component integrated into the homepage
- **Contact page with form and business info** - Hours, location, social links, and a styled contact form
- **Responsive, no component libraries** - All layout and styling via CSS Modules and CSS custom properties. No Tailwind, no Bootstrap, no UI kits

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **UI:** React 19
- **Styling:** CSS Modules + CSS custom properties
- **SVG Generation:** Custom Node.js scripts (`generate-waves.js`, `generate-progress-line.js`)
- **Deployment:** Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Generate SVG assets (waves and progress line)
node generate-waves.js
node generate-progress-line.js

# Start development server
npm run dev
```

The dev server runs at `http://localhost:3000`.

## Build

```bash
npm run build
npm start
```
