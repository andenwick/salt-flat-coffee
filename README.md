# SaltFlat Coffee

Marketing and e-commerce site for a small-batch artisan coffee roaster based in Tooele, Utah.

## Features

- Hero with custom scroll animations and SVG wave section dividers
- Product catalog and shop pages
- About and contact pages
- Newsletter signup
- Responsive design, no component libraries

## Stack

- Next.js 16 (App Router)
- React 19
- CSS Modules
- Custom SVG wave generators (`generate-waves.js`, `generate-progress-line.js`)

## Run locally

```bash
npm install
npm run dev
```
